Machine-failure-prediction


Objective
Predict whether an industrial machine operation will fail or not using machine learning.

Dataset Features
- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear
- Machine Type

Preprocessing
- Removed unnecessary ID columns
- Encoded categorical columns
- Split data into train and test sets

Model Used
Random Forest Classifier

Evaluation Metrics
- Accuracy Score
- Precision
- Recall
- F1-score
- Confusion Matrix

Conclusion
The model achieved high accuracy in predicting machine failures.

Future Improvements
- Hyperparameter tuning
- Cross-validation
- More feature engineering
